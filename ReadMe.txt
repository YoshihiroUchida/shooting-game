プレイ方法

1. Shooting Game.exeファイルを開きます。

2. タイトル画面が出るので、スペースキーで次の画面へ、バックスペースキーでこれまでのプレイヤーのスコアランキングを見ることができます。

3. プレイヤーの名前を入力します。

4. ステージを選択します。 (V：レベルを下げる ／ N：レベルを上げる)

5. 矢印キー or WASDキー で動かすことができます。Bで弾を発射できます。スペースキーでゲーム開始です。

6. プレイ中は、流れてくるアイテムを拾うことで、発射できる弾の種類を増やすことができます。V or N で切り替えができます。

7. 最後の敵を倒すと、ゲームクリアです。体力が0になるとゲームオーバーとなります。
